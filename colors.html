State Colors

Metronic comes with a number of state colors that can be applied to the most of elements and components. It reuses Bootstrap's original 6 states:

State Preview Class postfix Usage example
Success *-success btn-success
Warning *-warning btn-warning
Danger *-danger btn-danger
Info *-info btn-info
Primary *-primary btn-primary
Secondary *-secondary btn-secondary

And also adds its own brand and metal states

State Preview Class addon Usage example
Brand *-brand btn-success
Accent *-accent btn-accent
Focus *-focus btn-focus
Metal *-metal btn-metal
Light *-light btn-light
