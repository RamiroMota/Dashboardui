Diseño Responsivo

Tamaños responsivos de Frow

Frow usa cuatro tamaños como puntos de corte en la grid. Cada uno está diseñado para cubrir un tipo de dispositivo.

                
    XS - ( Pantallas    0 px - 767 px  ancho ) - General Telefonos / Tablets pequeñas
    SM - ( Pantallas  768 px - 991 px  ancho ) - General Tablets
    MD - ( Pantallas  992 px - 1199 px ancho ) - General Computadoras / Tabletas de paisaje
    LG - ( Pantallas 1200 px +         ancho ) - General Ordenadores con pantalla grande

Apilamiento de columna móvil

En las pantallas XS y SM, las fracciones de la columna se tratan automáticamente como 1-1. Estas tres columnas se apilarán una encima de la otra en el móvil de forma predeterminada, sin necesidad de código adicional. Esto evita que su contenido quede demasiado comprimido en los teléfonos.


    <div class="frow">
        <div class="col-md-1-3"></div> // col-xs-1-1 and col-sm-1-1 are assumed
        <div class="col-md-1-3"></div> // col-xs-1-1 and col-sm-1-1 are assumed
        <div class="col-md-1-3"></div> // col-xs-1-1 and col-sm-1-1 are assumed
    </div>
                
                // Reducir la ventana de su navegador para ver la pila de columnas como si fueran 1-1 en lugar de 1-3
                // Expande la ventana de tu navegador para ver que estas aparentemente 1-1 columnas aparecen como 1-3

Clases Responsivas

A las columnas también se les pueden dar fracciones .col-sm y .col-xs para anular el cambio a 1-1 en los dispositivos móviles. Por ejemplo, si desea que tres columnas permanezcan como tres columnas en los dispositivos móviles, simplemente defina las clases .col-xs y .col-md.


    <div class="frow">
        <div class="col-xs-1-3 col-sm-1-3 col-md-1-3"></div>
        <div class="col-xs-1-3 col-sm-1-3 col-md-1-3"></div>
        <div class="col-xs-1-3 col-sm-1-3 col-md-1-3"></div>
    </div>

Después de esa cuadrícula básica, veamos dónde se destaca Frow.

Si las fracciones suman menos que un todo, Frow las centra. No me gusta? Una clase rápida cambia el comportamiento. Vea todas las clases disponibles en los accesos directos de Frow.

                        
    <div class="frow"> // Centrar columnas horizontalmente por defecto

.col-md-3-5

                    
    <div class="frow frow-start"> // Cambia a la alineación típica a la izquierda

.col-md-3-5

                    
    <div class="frow row-end"> // Cambia a la alineación a la derecha

.col-md-3-5
